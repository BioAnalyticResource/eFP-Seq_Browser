eFP-Seq Browser User Data Submission

To visualize and interpret your own RNA-seq data with the eFP-Seq Browser, please fill out the form below with the informationfor your data. All fields with an * are required fields.

When you have finished filling in the forms, please click the "Generate XML" button at the bottom of the page. It will give you an option to download an XML file of readable data; it is highly recommended that you download that XML file so you may simply submit that XML file next time instead of needing to fill out the form again. You can also choose to make the XML file publicly accessible

If you have a lot of RNA-Seq data files to input into the eFP-Seq Browser, it may be best to upload an Excel file with all the information filled in instead of typing it below. You can download the Excel template here
